Breast Cancer Prediction

This project involves building a machine learning model to predict breast cancer using a Jupyter Notebook. The model is developed using the scikit-learn library and employs the Decision Tree algorithm for classification. The dataset used is the Breast Cancer Wisconsin (Diagnostic) dataset, which includes features computed from a digitized image of a fine needle aspirate (FNA) of a breast mass. The goal is to accurately classify whether a tumor is benign or malignant based on these features.
